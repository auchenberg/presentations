Core protocols

  • SDP - Session Description Protocol

    Format for describing streaming media initialization parameters

  • STUN - Session Traversal Utilities for NAT

    Enables announcing your public IP addrreess during the negotiation phase of a session

  • TURN - Traversal Using Relays around NAT

    Relays all media via a TURN server that is visible to all parties in a session

  • ICE - Interactive Connectivity Establishment

    Decide the best solution connect (direct, STUN or TURN)
